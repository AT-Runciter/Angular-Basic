Створила фейковий API, використавши json-server.

Поетапне створенння:

1) npm install -g json-server - встановлення через консоль;

2) створення файлу db.json зі структурою, яка буде в фейковому API;

3) команда в консолі json-server --watch db.json - запуск json-server з db.json

Після запуску json-server запускає фейковий API на локальному хості на порту 3000 (http://localhost:3000/exercises).
